д»»дҪ•жәҗд»Јз Ғж–Үд»¶еҝ…йЎ»еұһдәҺжҹҗдёӘеҢ…пјҢеҗҢж—¶жәҗз Ғж–Үд»¶зҡ„з¬¬дёҖиЎҢжңүж•Ҳд»Јз Ғеҝ…йЎ»жҳҜpackage pacakgeName иҜӯеҸҘпјҢйҖҡиҝҮиҜҘиҜӯеҸҘеЈ°жҳҺиҮӘе·ұжүҖеңЁзҡ„еҢ…гҖӮ

еҢ…зҡ„еҹәжң¬жҰӮеҝө

GoиҜӯиЁҖзҡ„еҢ…еҖҹеҠ©дәҶзӣ®еҪ•ж ‘зҡ„з»„з»ҮеҪўејҸпјҢдёҖиҲ¬еҢ…зҡ„еҗҚз§°е°ұжҳҜе…¶жәҗж–Үд»¶жүҖеңЁзӣ®еҪ•зҡ„еҗҚз§°пјҢиҷҪз„¶GoиҜӯиЁҖжІЎжңүејәеҲ¶иҰҒжұӮеҢ…еҗҚеҝ…йЎ»е’Ңе…¶жүҖеңЁзҡ„зӣ®еҪ•еҗҚеҗҢеҗҚпјҢдҪҶиҝҳжҳҜе»әи®®еҢ…еҗҚе’ҢжүҖеңЁзӣ®еҪ•еҗҢеҗҚпјҢиҝҷж ·з»“жһ„жӣҙжё…жҷ°гҖӮ

  • еҢ…еҗҚдёҖиҲ¬жҳҜе°ҸеҶҷзҡ„пјҢдҪҝз”ЁдёҖдёӘз®Җзҹӯдё”жңүж„Ҹд№үзҡ„еҗҚз§°гҖӮ
  • еҢ…еҗҚдёҖиҲ¬иҰҒе’ҢжүҖеңЁзҡ„зӣ®еҪ•еҗҢеҗҚпјҢд№ҹеҸҜд»ҘдёҚеҗҢпјҢеҢ…еҗҚдёӯдёҚиғҪеҢ…еҗ«- зӯүзү№ж®Ҡз¬ҰеҸ·гҖӮ
  • еҢ…дёҖиҲ¬дҪҝз”ЁеҹҹеҗҚдҪңдёәзӣ®еҪ•еҗҚз§°пјҢиҝҷж ·иғҪдҝқиҜҒеҢ…еҗҚзҡ„е”ҜдёҖжҖ§пјҢжҜ”еҰӮ GitHub йЎ№зӣ®зҡ„еҢ…дёҖиҲ¬дјҡж”ҫеҲ°GOPATH/src/github.com/userName/projectName зӣ®еҪ•дёӢгҖӮ
  • еҢ…еҗҚдёә main зҡ„еҢ…дёәеә”з”ЁзЁӢеәҸзҡ„е…ҘеҸЈеҢ…пјҢзј–иҜ‘дёҚеҢ…еҗ« main еҢ…зҡ„жәҗз Ғж–Үд»¶ж—¶дёҚдјҡеҫ—еҲ°еҸҜжү§иЎҢж–Үд»¶гҖӮ
  • дёҖдёӘж–Үд»¶еӨ№дёӢзҡ„жүҖжңүжәҗз Ғж–Үд»¶еҸӘиғҪеұһдәҺеҗҢдёҖдёӘеҢ…пјҢеҗҢж ·еұһдәҺеҗҢдёҖдёӘеҢ…зҡ„жәҗз Ғж–Үд»¶дёҚиғҪж”ҫеңЁеӨҡдёӘж–Үд»¶еӨ№дёӢгҖӮ
